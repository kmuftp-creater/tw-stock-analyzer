系統說明與教學手冊

📌 系統特色與核心功能

  1. 多因子綜合評分 (防洗盤)： 勾選「適合買進」時，系統會啟動防護，強制要求股價必須站上 MA20 月線，且多項指標加總得分需 ≥ 2，幫您過濾掉多數的盤整假突破。
  2. API 競速抓取技術： 點擊「開始篩選」後，系統不需後端，而是透過前端代理程式(Proxy)平行競速，即時向 Yahoo Finance 與 FinMind 等多個開源 API 抓取歷史 K 線，確保數據即時且連線不易中斷。
  3. 大盤多空位階濾網 (Market Regime Filter)： 系統會在背景默默分析加權指數。當大盤跌破月線或 RSI < 40 時，代表整體市場轉弱。此時系統會自動將所有個股的「強烈買進」建議封鎖，降級為「保守進場」，避免您在空頭市場中受傷。
  4. 情緒溫度計： 頂部大盤旁的進度條，透過大盤 RSI 視覺化顯示目前市場情緒。綠色代表恐懼(通常是反彈買點)，紅色代表貪婪(隨時可能回檔)。
  5. AI 純前端診斷： 在篩選結果表格中點擊「AI 診斷」，系統會根據該股票滿足的條件，自動組合出一句白話文報告，解釋為什麼會給出這樣的操作建議。

💡 實戰範例教學

【範例一】尋找強勢突破股： 勾選「適合買進」+「量能異動」+「布林突破上軌」。
👉 適用時機：大盤處於多頭(偏多)時，尋找剛帶量發動的飆股。

【範例二】搶反彈策略 (左側交易)： 勾選「超跌反彈」+「IBS ≤ 0.2」+「跌破 MA20」。
👉 適用時機：個股短線乖離過大，尋找收在當日最低點且 RSI 極低的標的，博取隔日反彈。

📈 內建技術指標原理

  • 均線 (SMA)：平滑價格雜訊。站上短均線代表轉強；跌破長均線代表轉弱。
  • 相對強弱指標 (RSI)：計算漲跌幅比例。跌破 30 為超賣(過冷)；突破 70 為超買(過熱)。
  • MACD 指標：利用長短 EMA 聚合分離判斷動能。紅柱代表動能向上。
  • 布林通道 (Bollinger Bands)：加上標準差反映股價波動合理區間。突破上軌通常伴隨大行情。
  • KD 隨機指標：比較收盤價與近期高低點相對位置，反映極短線強弱。
